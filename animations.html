Error Handling - MediaDevices API

The camera system handles three specific error types from the MediaDevices API:

NotAllowedError

User denied camera permission. Message: "Camera permission denied. Please allow camera access in settings."

NotFoundError

No camera hardware detected. Message: "No camera device found. Check your hardware."

NotReadableError

Camera is in use by another application. Message: "Camera is already in use by another application."

Canvas Image Processing

Greyscale Filter

ITU-R BT.601 luminance formula: Y = 0.299R + 0.587G + 0.114B Green weighted highest because human eyes perceive green most sensitively.

Sepia Effect

Historical photo tone using weighted RGB transformation. Creates warm, vintage appearance by emphasizing red and yellow channels.

Invert Filter

Simple RGB inversion: R' = 255 - R, preserves alpha channel. Demonstrates basic pixel manipulation on Uint8ClampedArray.
